# Security Policy

## Reporting Security Issues

Please report security concerns privately to contact@thequantum.foundation.

Do not open a public GitHub issue for active vulnerabilities, private keys, deployment secrets, or exploit details.

## Sensitive Information

Never commit private keys, mnemonics, RPC credentials, GitHub passwords, personal access tokens, or production environment files. Use local `.env` files, wallet tooling, hardware wallets, multisigs, and credential managers instead.
